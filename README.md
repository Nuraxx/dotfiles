# Dotfiles

My personal Arch Linux + Hyprland dotfiles.

## Setup

- WM: Hyprland
- Bar: Waybar
- Notifications: SwayNC
- Terminal: WezTerm
- Launcher: Wofi
- Shell: Zsh
- Wallpaper: Hyprpaper
- Fetch: Fastfetch
